Documentación de la API Flask

Descripción

Esta API Flask permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) en una colección de productos almacenados en una base de datos MongoDB.

Recursos Disponibles

1. Obtener todos los productos

URL: `/`

Método HTTP: GET

Descripción: Obtiene todos los productos almacenados en la base de datos.

Respuesta Exitosa: Devuelve una lista de productos en formato JSON.

Códigos de Respuesta: 200 (Éxito)

2. Agregar un nuevo producto

URL: `/products`

Método HTTP: POST

Descripción: Agrega un nuevo producto a la base de datos.

Datos de Entrada: Nombre (`name`), Precio (`price`), Cantidad (`quantity`).

Respuesta Exitosa: Devuelve los detalles del producto agregado en formato JSON.

Códigos de Respuesta: 200 (Éxito), 400 (Solicitud Incorrecta)

3. Eliminar un producto

URL: `/delete/`

Método HTTP: GET

Descripción: Elimina un producto de la base de datos por su nombre.

Respuesta Exitosa: Redirige a la página de inicio después de la eliminación.

Códigos de Respuesta: 302 (Redirección)

4. Editar un producto

URL: `/edit/`

Método HTTP: POST

Descripción: Edita un producto en la base de datos por su nombre.

Datos de Entrada: Nombre (`name`), Precio (`price`), Cantidad (`quantity`).

Respuesta Exitosa: Redirige a la página de inicio después de la edición.

Códigos de Respuesta: 302 (Redirección), 400 (Solicitud Incorrecta)

Errores Comunes

Código de Respuesta: 404 (No Encontrado)

Descripción: La página solicitada no existe.

Código de Respuesta: 400 (Solicitud Incorrecta)

Descripción: La solicitud no cumple con los requisitos necesarios.
